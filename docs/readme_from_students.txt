Key Features:
1. Login
2. Check Login user type and display functionality buttons accordingly.
3. Add New Library Member
4. Edit Library Member Information
5. Add Book (extra use case)
6. Add Book Copy
7. Check Out Book
8. Check In Book (extra use case)

Folder Structure
1. \application - Main class and application css.
2. \business - Domain classes
3. \dataaccess - Classes for data access layer.
4. \docs - UML(class, sequence) diagrams, presentation file.
5. \ui - FXML objects, controller classes and images.
